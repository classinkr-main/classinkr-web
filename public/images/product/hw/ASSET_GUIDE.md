# ClassIn Board HW 제품 페이지 — 이미지 에셋 가이드

## 폴더 구조
```
public/images/product/hw/
├── hero/           → 히어로 섹션 (전자칠판 메인 비주얼)
├── board/          → 제품 자체 사진 (앞면, 측면, 디테일)
├── writing/        → 필기 체험 섹션 (판서 장면, 표면 클로즈업)
├── display/        → 디스플레이 품질 (시야각, 코팅 비교)
├── sharing/        → 판서 공유 (보드→디바이스 연결 장면)
├── camera/         → AI 카메라 & 마이크 (하이브리드 수업)
├── ecosystem/      → SW 생태계 통합 (ClassIn UI on Board)
├── lineup/         → 라인업 비교 (4모델 사이즈 비교)
└── spaces/         → 설치 공간별 (교실, 강당, 회의실, 스터디룸)
```

## 네이밍 컨벤션

```
{section}-{subject}-{variant?}.{ext}
```

- section: hero, board, writing, display, sharing, camera, ecosystem, lineup, space
- subject: 핵심 대상 (board, teacher, tablet, flow 등)
- variant: 선택 (compare, detail, angle 등)
- ext: .png (투명배경/UI), .jpg (사진/장면), .svg (다이어그램)

## 이미지 포맷 가이드

| 유형 | 포맷 | 이유 |
|------|------|------|
| 제품 컷아웃 (투명 배경) | PNG | 배경 투명 필요 |
| 교실/사용 장면 | JPG (80%) | 용량 최적화 |
| 다이어그램/플로우 | SVG or PNG | 벡터 선명도 |
| 히어로 배경 | JPG + next/image | blur placeholder |
